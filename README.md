# P14-HRNet

### Demarer l'app
 
1. Lancer le projet depuis VSCode
2. Dans un terminal de commande VSCode, réaliser la suite de commandes :
3. - `npm i`
   - `npm run dev`
4. Se rendre sur : `http://localhost:5173/`
----

### Version build
1. Lancer le projet depuis VSCode
2. Dans un terminal de commande VSCode, réaliser la suite de commandes :
3. - `npm i`
   - `npm run build`
   - puis `npm run preview`
4. Se rendre sur : `http://localhost:4173/`
